# 项目介绍

基于Linux的轻量级多线程网络服务器。

### 网络服务器

- 一个服务器软件（程序），或者是运行这个服务器软件的硬件（计算机）。
- 功能：通过HTTP协议与客户端（通常是浏览器）进行通信，来接收，存储，处理来自客户端的HTTP请求，并对其请求做出HTTP响应，返回给客户端其请求的内容（文件、网页等）或返回一个Error信息。

### 用户与网络服务器的通信方式

- 通常用户使用web浏览器与相应服务器进行通信。
- 在浏览器中键入“域名” 或 “IP地址：端口号” 。
